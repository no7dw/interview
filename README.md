要求：以下题目答题是30分钟,旨在考察面试者多方面的能力。请尽可能多答，算法题可以任何编程语言的代码（欢迎加关键注释），也可以写出大概想法（得分会减少）。

我们[公司][1]每周3都会举行康体活动，一般是分成几组进行对抗。
现在想设计一个算法，来进行随机分组，假设：
员工信息（名字）都保存在一个数组里面 Array[a1,a2,a3,a4…an]，分组数目为k。

 - 代码写出算法，

 - 为其编写尽量多unit test
